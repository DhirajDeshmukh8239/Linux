# $$\color{blue}\textbf{ User Management}$$
### Types Of User:
 - 1.Root user (full permission)                                 id=0
 - 2.System user (created by system and has no login shell)      id=1-999
 - 3.Local user (limited permission)                             id=1000-65535

## Commands

**switch to root user**
````
sudo -i
````
**check user id**
````
id user (example: id root)
````
**User information**
````
cat /etc/passwd
````
